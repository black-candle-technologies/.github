# Black Candle Technologies

We build developer tools that make AI and data infrastructure faster, cheaper, and more resilient.

## Flagship Product

### Chitin
A hosted AI gateway that optimizes every token — caching, semantic deduplication, context compression, and intelligent model routing. Zero infrastructure required.

Chitin is a cloud-hosted proxy that sits between your applications and upstream LLM providers. Change one URL in your code and immediately get ≥40% token savings across all your LLM calls.

## Projects

### [Wayfinder](https://github.com/black-candle-technologies/wayfinder)
An offline-first data layer and caching runtime for Next.js apps. Wayfinder provides seamless offline experiences, automatic mutation queueing, background sync, and a plugin-based architecture for integrating with external data sources.

### [Lumen](https://github.com/black-candle-technologies/lumen)
A local-first AI agent that lives inside your chat apps. Private, persistent, and extensible — Lumen runs on your device without sending your data to the cloud.
